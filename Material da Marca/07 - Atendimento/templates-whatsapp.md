# Templates de Atendimento - WhatsApp Movelart

Mensagens prontas para as situações mais comuns. Adapte sempre ao contexto - nunca copie e cole sem ler antes.

```
Olá, [Nome]!

Preparei a proposta para o seu [ambiente]. Está anexada aqui.

O projeto inclui:
- [Item 1]
- [Item 2]
- [Item 3]

O prazo estimado de produção e instalação é de [X a Y semanas].

Fica à vontade para revisar com calma. Posso tirar qualquer dúvida que surgir. Tem algum horário bom esta semana para conversarmos sobre os detalhes?
```

## Follow-up após envio de proposta (sem resposta em 3–5 dias)

```
Olá, [Nome]! Tudo bem?

Passei para saber se teve a oportunidade de ver a proposta que enviei para o [ambiente].

Qualquer ajuste ou dúvida, pode me chamar - gosto de entender o que faz mais sentido para cada projeto antes de fechar.
```

```

Vou analisar o caso com atenção e entro em contato até [prazo] com uma solução. Nosso compromisso é com a qualidade e a satisfação do cliente em cada etapa - e isso inclui o pós-entrega.

Obrigado por me avisar. Vamos resolver.
```

---

## Boas práticas de atendimento

- **Sempre use o nome do cliente** - humaniza e cria vínculo
- **Primeira resposta em até 2h** em dias úteis (das 8h às 18h)
- **Não use áudio** para orçamentos ou informações importantes - prefira texto para o cliente consultar depois
- **Confirme o entendimento** antes de propor soluções: "Se entendi certo, você quer..."
- **Finalize sempre com uma pergunta ou ação clara** - evita o silêncio e mantém a conversa fluindo
