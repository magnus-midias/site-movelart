# logos/

Armazena os arquivos de logotipo do projeto.

## Formatos aceitos
- SVG (preferencial — escalável e leve)
- PNG com fundo transparente (para contextos que não suportam SVG)

## Nomenclatura
```
logo-<variante>-<tema>.ext
```

Exemplos:
- `logo-principal-claro.svg`
- `logo-principal-escuro.svg`
- `logo-icone-claro.svg`
- `logo-icone-escuro.png`

## Regras
- Sem acentos ou caracteres especiais nos nomes de arquivo.
- Usar kebab-case.
- Manter versão clara e escura de cada variante.
