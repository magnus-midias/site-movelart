# favicon/

Armazena o favicon e suas variantes para diferentes plataformas.

## Arquivos recomendados
| Arquivo | Tamanho | Uso |
|---|---|---|
| `favicon.ico` | 32×32 | Navegadores legados |
| `favicon-32x32.png` | 32×32 | Navegadores modernos |
| `favicon-16x16.png` | 16×16 | Aba do navegador |
| `apple-touch-icon.png` | 180×180 | iOS / Safari |
| `android-chrome-192x192.png` | 192×192 | Android |
| `android-chrome-512x512.png` | 512×512 | Android splash |
| `site.webmanifest` | — | PWA manifest |

## Regras
- Sem acentos ou caracteres especiais nos nomes de arquivo.
- Usar kebab-case.
- Fundo sólido ou transparente — evitar gradientes complexos em tamanhos pequenos.
