# icones/

Armazena ícones customizados do projeto (além de qualquer biblioteca de ícones usada via pacote).

## Formatos aceitos
- SVG (preferencial)
- PNG com fundo transparente

## Nomenclatura
```
icone-<nome>-<tamanho>.ext
```

Exemplos:
- `icone-seta-direita-24.svg`
- `icone-notificacao-32.svg`

## Regras
- Sem acentos ou caracteres especiais nos nomes de arquivo.
- Usar kebab-case.
- Incluir o tamanho em px no nome quando houver múltiplas versões.
- SVGs devem ter `viewBox` definido e sem dimensões fixas em `width`/`height`.
